# Blipp AI Core Engine

The Intelligence Layer for Modern States & Organizations. Secure, integrated, reliable AI for governments and enterprises.

## Technologies

- **Vite**: Next Generation Frontend Tooling
- **TypeScript**: Typed JavaScript at Scale
- **React**: A JavaScript library for building user interfaces
- **shadcn/ui**: Beautifully designed components
- **Tailwind CSS**: A utility-first CSS framework

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

```sh
# Install dependencies
npm install
```

### Development

```sh
# Start development server
npm run dev
```

### Build

```sh
# Build for production
npm run build
```

## Features

- **Intelligence Layer**: Embeds AI directly into organizational workflows.
- **Enterprise Ready**: Built for reliable real-world applications.
- **Secure & Integrated**: Designs with security and integration in mind.
